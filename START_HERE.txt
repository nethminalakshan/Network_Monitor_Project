┌───────────────────────────────────────────────────────────────────────────────┐
│                      🎮 QUICK USAGE GUIDE 🎮                                 │
└───────────────────────────────────────────────────────────────────────────────┘

  Step 1: Run start-app.bat
  
  Step 2: Wait for applications to start
          • Backend API (port 8080)
          • Frontend UI (port 3000)
  
  Step 3: Browser opens automatically to http://localhost:3000
  
  Step 4: Select scan mode:
          • Quick Scan - Monitor local + DNS (fastest, recommended)
          • Custom IPs - Enter specific addresses
          • Full Network Scan - Scan entire subnet
  
  Step 5: Click "▶️ Start Monitoring"
  
  Step 6: Watch real-time data appear!
  
  Step 7: Click "⏹️ Stop Monitoring" when done

┌───────────────────────────────────────────────────────────────────────────────┐
│                    🛠️ TROUBLESHOOTING 🛠️                                    │
└───────────────────────────────────────────────────────────────────────────────┘

  Problem: "Java not found"
  Solution: Install Java JDK from https://www.oracle.com/java/technologies/downloads/
  
  Problem: "Node not found"
  Solution: Install Node.js from https://nodejs.org/
  
  Problem: "Port 8080 already in use"
  Solution: Close other applications using port 8080
  
  Problem: "Failed to fetch monitoring data"
  Solution: Make sure backend is running (check terminal window)
  
  Problem: "No devices found"
  Solution: Run as Administrator and try Quick Scan mode first
